# 回归分析网页应用 — 技术规格书

> 版本: v1.1 | 日期: 2026-07-15 | 状态: 已确认，待实现

---

## 1. 项目概述

### 1.1 一句话描述
一个纯前端单 HTML 文件应用：用户拖放 CSV 文件，选择因变量和自变量后，自动完成回归分析，在页面上展示完整统计结果与诊断图表。

### 1.2 核心价值
- **零安装零部署**：一个 HTML 文件，浏览器打开即用
- **本地计算**：所有数据在浏览器端处理，不上传服务器，数据安全
- **统计专业**：覆盖 OLS 线性回归 + 逻辑回归，含完整诊断指标

### 1.3 适用场景
- 快速探索性数据分析
- 教学演示（回归分析概念）
- 小规模数据的回归建模（建议 < 10 万行）

## 2. 用户操作流程

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 拖放CSV  │───▶│ 预览数据  │───▶│ 选择变量  │───▶│ 回归分析  │───▶│ 结果展示  │
│   文件   │    │ 检查格式  │    │ 因变量/Y  │    │ 自动计算  │    │ 表格+图表 │
└──────────┘    └──────────┘    │ 自变量/X  │    └──────────┘    └──────────┘
                                └──────────┘
```

### 步骤详解

**Step 1 — 导入数据**
- 页面中央显示拖放区域，支持点击选择文件
- 仅接受 `.csv` 文件
- 使用 Papa Parse 解析，自动检测分隔符（逗号/制表符/分号）
- 自动识别编码（UTF-8 / GBK / GB2312）

**Step 2 — 数据预览**
- 以表格形式展示前 20 行数据
- 每列显示：列名、数据类型（数值/文本/日期）、缺失值数量
- 高亮标记非数值列（回归分析需要数值类型）

**Step 3 — 变量选择**
- 下拉选择因变量（Y），仅显示数值型列
- 多选/复选框选择自变量（X1, X2, ...），仅显示数值型列
- 自动检测因变量是否为二分类变量（提示用户可选择逻辑回归）
- 可选：是否包含截距项（默认包含）

**Step 4 — 执行分析**
- 点击「开始分析」按钮
- 自动判断回归类型：
  - 因变量连续 → 多元线性回归 (OLS)
  - 因变量二分类 → 逻辑回归 (Logistic)
- 处理缺失值：默认删除含缺失值的行，提示删除行数

**Step 5 — 查看结果**
- 结果分区展示，支持折叠/展开
- 表格展示 + 图表可视化

## 3. 技术架构

### 3.1 技术选型

| 层级 | 技术 | 说明 |
|------|------|------|
| CSV 解析 | Papa Parse 5.x (CDN) | 最成熟的 JS CSV 解析库 |
| 矩阵运算 | ml-matrix (CDN) | 回归计算核心依赖 |
| 线性回归 | 自实现 (基于 ml-matrix) | OLS: β = (X'X)⁻¹X'y |
| 逻辑回归 | 自实现 (梯度下降) | 二分类，Sigmoid + 交叉熵损失 |
| 统计分布 | jStat (CDN) | t/F/正态分布 CDF，用于 p 值计算 |
| 可视化 | Chart.js 4.x (CDN) | 散点图、残差图、QQ 图、系数图 |
| UI 框架 | 无框架，Vanilla CSS | 保持零依赖，轻量 |
| 图标 | 内联 SVG | 避免额外 CDN 依赖 |

### 4.1 CSV 导入模块 (csvImport)

```

  - 文件类型校验 (.csv only)

    headers: string[],        // 列名数组
    data: object[],           // 解析后的数据对象数组
    rowCount: number,         // 总行数
    colTypes: {name: type}[], // 每列的数据类型
    missingCount: {name: n}   // 每列缺失值数量
  }
```

### 4.2 回归计算引擎 (regressionEngine)

#### 4.2.1 OLS 多元线性回归

```
核心公式：
  β = (X'X)⁻¹ X'y

计算步骤：
  1. 构建设计矩阵 X（n × (k+1)），第一列全 1（截距项）
  2. 提取因变量向量 y（n × 1）
  3. 删除含 NaN 的行，记录删除行数
  4. 计算 X'X，判断是否可逆（条件数检查）
  5. 求解 β = (X'X)⁻¹ X'y
  6. 计算预测值 ŷ = Xβ
  7. 计算残差 e = y - ŷ

产出指标：
  ┌─────────────────────────────────────────────────┐
  │ 模型摘要                                         │
  │  R², 调整 R², F 统计量, F 的 p 值,              │
  │  残差标准误 (RSE), 观测数 n                      │
  ├─────────────────────────────────────────────────┤
  │ 系数表                                           │
  │  系数值, 标准误, t 值, p 值, 95% 置信区间        │
  │  (每个自变量 + 截距项)                           │
  ├─────────────────────────────────────────────────┤
  │ 模型诊断                                         │
  │  VIF (方差膨胀因子) — 多重共线性诊断             │
  │  Durbin-Watson — 自相关诊断                      │
  │  Jarque-Bera / 偏度峰度 — 残差正态性             │
  │  Breusch-Pagan — 异方差检验(简化版)              │
  └─────────────────────────────────────────────────┘
```

**VIF 计算**：
```
VIF_j = diag[(X'X)⁻¹]_j
（对于包含截距项的设计矩阵，VIF_j 即为 (X'X)⁻¹ 对角线元素）
判定：VIF > 10 表示严重多重共线性
```

**Durbin-Watson**：
```
DW = Σ(e_t - e_{t-1})² / Σ(e_t)²
判定：≈ 2 无自相关，< 1 正自相关，> 3 负自相关
```

**Breusch-Pagan（简化版）**：
```
1. 以残差平方 e² 为因变量，原自变量为自变量做辅助回归
2. BP = n × R²_aux
3. BP 服从 χ²(k) 分布
判定：p < 0.05 表示存在异方差
```

#### 4.2.2 逻辑回归

```
触发条件：因变量只有 2 个唯一值（或用户手动选择）

算法：梯度下降 (Gradient Descent)
  - 模型：P(y=1|X) = 1 / (1 + e^{-Xβ})
  - 损失函数：交叉熵 (Cross-Entropy)
  - 正则化：可选 L2 正则化（默认关闭）
  - 收敛条件：损失变化 < 1e-6 或达到最大迭代次数(1000)

产出指标：
  - 系数, 标准误, z 值, p 值, 95% 置信区间
  - 似然比检验 (Likelihood Ratio Test)
  - Pseudo R² (McFadden)
  - 混淆矩阵 (默认阈值 0.5)
  - 准确率, 精确率, 召回率, F1
  - ROC AUC（可选，需计算完整 ROC 曲线）
```

## 6. 边缘情况与错误处理

| 场景 | 处理方式 |
|------|----------|
| 空文件 / 无数据行 | 提示「文件为空或无有效数据」 |
| 所有列都是非数值 | 提示「未检测到数值型列，无法回归」 |
| 自变量与因变量相同 | 阻止选择（下拉排除已选列） |
| 未选任何自变量 | 按钮置灰，提示「请至少选择一个自变量」 |
| 样本量过小 (n < 10) | 警告「样本量过小，结果可能不可靠」 |
| 完美多重共线性 | 捕获矩阵不可逆错误，提示用户去掉冗余变量 |
| 自变量含常量列 | 检测方差为 0 的列并自动排除 |
| 缺失值 | 默认删除含缺失行，提示删除数量 |
| 因变量非二分类选逻辑回归 | 警告「因变量非二分类，逻辑回归可能不适用」 |
| 数据全部分到同一类 (逻辑回归) | 提示「因变量只有一个类别，无法进行逻辑回归」 |
| 编码检测失败 | 弹出编码选择器（UTF-8 / GBK / GB2312），用户手动选择后重新解析 |
| 文件过大 (>50MB) | 提示文件过大，建议使用专业工具 |

## 附录 A：关键公式参考

### OLS 系数估计
```
β̂ = (XᵀX)⁻¹ Xᵀy
Var(β̂) = σ̂² (XᵀX)⁻¹
σ̂² = eᵀe / (n - k - 1)
```

### t 检验
```
t_j = β̂_j / SE(β̂_j)
p_j = 2 × (1 - F_t(|t_j|, n-k-1))
```

### 置信区间
```
CI_j = β̂_j ± t_{α/2, n-k-1} × SE(β̂_j)
```

### 逻辑回归梯度下降
```
β ← β - α × Xᵀ(y - ŷ) / n
ŷ = sigmoid(Xβ) = 1 / (1 + e^{-Xβ})
```
